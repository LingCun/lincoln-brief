> 1. 매일 같은 시간에 발송되었는가?
> 2. 글의 톤·깊이가 일관되었는가?
> 3. "추천" 표현이 한 번도 없었는가?

## 막힌 곳 우선 처리

진행 중 문제 발생 시:

| 증상 | 1차 점검 | 2차 |
|---|---|---|
| Stibee 다크 배경 깨짐 | Gmail 모바일 다크 모드 토글 확인 | 라이트 톤 버전 별도 제작 |
| SVG 썸네일 미표시 | PNG 변환 (Sharp/Puppeteer) | 새 썸네일은 처음부터 PNG로 |
| 자동화 발행 실패 | Task Scheduler 로그 | Claude CLI 인증 만료 확인 |
| 발송 시간 어긋남 | Windows 시계 동기화 | NTP 서버 변경 |
| 구독자 정체 | 콘텐츠 자체 검토 (가치 명확한가) | 마케팅 채널 1개 추가 |
