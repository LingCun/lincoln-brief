# Content Model — 무료/유료 분리 규칙

> Lincoln Brief 무료 vs 유료 콘텐츠의 정확한 차이. 모든 발행 글이 이 규칙을 따른다.
>
> 전략·차별화 맥락은 [STRATEGY.md](STRATEGY.md), 톤·문체 규칙은 루트 STYLE.md 가 단일 출처.

---

## 1. 단일 소스 원칙

**한 글은 하나의 markdown 파일에서 출발한다.** 무료/유료를 따로 쓰지 않음 — 운영 부담 폭증.

```
src/content/blog/foo-20260519.md
├─ frontmatter
│   ├─ title, description, pubDate, ...
│   └─ paywallAfterSection: "결정적인 한 줄"   ← 무료 사용자에게 보이는 마지막 섹션
│
└─ body (markdown)
    ├─ [무료 영역] 도입 + 데이터 표 + 문제 제기
    │   └─ paywallAfterSection 기준점
    ├─ ────────────  cut here  ────────────
    └─ [유료 영역] 분석 + 결론 + Lincoln의 한 마디 + 워치리스트
```

빌드 시 `paywallAfterSection` 이후를 잘라낸 버전이 사이트(무료) 페이지에 생성됨. 30일 후 풀버전 자동 공개.

---

## 2. 무료 영역에 반드시 들어가는 것

| 구성요소 | 이유 |
|---|---|
| 제목 + description | SEO + 유입 |
| 한 줄 요약 인용구 | 가치 신호 |
| 데이터 표 (핵심 사실) | "데이터는 풀어준다" — 신뢰 |
| 도입부 분석 1~2 단락 | 글의 톤·깊이 노출 |
| 문제 제기 / 질문 | **결제 동기 핵심** |
| 썸네일 SVG | 시각적 자산 |
| Sources (출처) | 신뢰 |

---

## 3. 유료 영역(=잘리는 부분)에 들어가는 것

| 구성요소 | 이유 |
|---|---|
| **결론·답** ("그래서 어떻게 봐야 하나") | 결제 가치 1순위 |
| **시사점** (한국 시장, 일반 투자자 의미) | 결제 가치 2순위 |
| **Lincoln의 한 마디** (필자 의견·전망) | 결제 가치 3순위 — 인물 신뢰 |
| **워치리스트** (당일 보고 있는 자리) | 결제 가치 4순위 — 액션 |
| 다음 30일 관찰 일정 | 추가 가치 |

---

## 4. 무료 사이트에서 잘리는 지점 — 예시 (UNH 글 기준)

**무료 사용자가 보는 것**:
```
✓ 제목: 버핏 없는 버크셔의 첫 카드
✓ 한 줄 요약 인용구
✓ "먼저 큰 그림부터" 섹션 (13F 설명 + 핵심 사실 표)
✓ "잠깐 — 왜 워런 버핏 답지 않은가요?" 콜아웃 박스
✓ "UNH의 9개월" 시간표 표
✓ 결정적인 한 줄 (1줄)
─────────── 여기서 잘림 ───────────
[잠금: 유료 회원 또는 30일 후 공개]
```

**유료 회원이 추가로 보는 것**:
```
+ 후계 체제의 첫 큰 카드 (Greg Abel 분석)
+ 13F의 45일 함정
+ 시장이 복사 매매 해도 될까 (단·중·장기 판단)
+ 다음 30일 관찰 일정
+ 한국 시장 시사점
+ 일반 투자자한테는 무슨 의미? (3가지 액션 가이드)
+ 한 줄 정리
+ Lincoln의 한 마디
+ [신규] Lincoln의 워치리스트
```

전체 글 분량의 **약 30%가 무료, 70%가 유료**.

## 5. 무료 영역 마지막에 들어갈 CTA

잘리는 지점에 자동으로 삽입되는 박스:

```markdown
─────────────────────────────────────
📬 매일 아침 6시, 5분 안에 답까지.

위 질문에 대한 Lincoln의 답과 워치리스트는
유료 구독자 메일함으로 06:00에 도착합니다.

[ 무료로 받아보기 → ]    [ 유료 구독 →  ]

* 30일 후 사이트에서 풀버전 무료 공개
─────────────────────────────────────
```

## 6. 워치리스트 섹션 표준 포맷

매일 메일 끝부분, 1~3개 종목:

```markdown
## Lincoln의 워치리스트 — 2026.05.19

### ◆ HUM (Humana) — 헬스케어 매니지드 케어
- 현재가: $268
- 보고 있는 자리: $250~$260
- 손절 라인: $240
- 익절 1차: $290 (한 달 +48% 모멘텀 둔화 신호)
- 왜: UNH 청산 → HUM·CI·ELV로 자금 이동 흐름.
  Forward PE 14.2로 동종 중 저평가.

### ◆ XLV (헬스케어 섹터 ETF)
- 현재가: $147
- 자리: 1년 +12%로 시장 절반. 비중 점검 대상.

> ※ 추천이 아닌 Lincoln이 관찰 중인 자리입니다.
> 투자 결정은 본인 책임으로 진행하세요.
```

**규칙**:
- "추천"이 아닌 **"보고 있는"** 표현만 사용 (법적 안전선)
- 손절가 명시 (전문성 신호)
- 진입 자리에 도달하면 다음 메일에 표시 ("◆ 진입 자리 도달")
- 청산 자리 도달 시 회고 ("XX에서 익절 — +X%")
- 익절·손절·청산 기록 누적 → 트랙 레코드 자산

## 7. 발행 일정 표준

| 시간 | 발행 | 내용 |
|---|---|---|
| 평일 06:00 KST | 미국 증시 브리프 (메일+사이트 무료) | 데일리시황 1편, 가끔 종목/예측/이슈 추가 |
| 평일 16:00 KST | 한국 증시 브리프 (메일+사이트 무료) | KRX 마감 +30분 후 |
| 주말 | 발송 없음 | 또는 주간 정리 1편 |
| 휴장일 | 발송 없음 | 사전 등록된 휴장일 표 자동 스킵 |

발송 1통 기준 — 위 4 카테고리를 다 묶지 않음. 매일은 1~2 카테고리 집중.

## 8. 면책 조항 (모든 발행 글 하단)

```
※ 본 글은 정보 제공 목적이며, 투자 권유나 종목 추천이 아닙니다.
※ 모든 투자 결정의 책임은 본인에게 있습니다.
※ Lincoln Brief는 한국 금융투자업법상 투자자문업 등록 사업자가 아닙니다.
```

## 9. 콘텐츠 품질 체크리스트

매 글 발송 전 확인:

- [ ] 데이터 출처 명시 (sources에 최소 2개)
- [ ] 추천 표현 없음 ("사라/팔아라/매수/매도" 단어 검색)
- [ ] Lincoln의 한 마디에 본인 의견 명시
- [ ] 워치리스트의 손절가 명시
- [ ] 면책 조항 포함
- [ ] 썸네일 카테고리 색 정합
- [ ] 평문 5분 안에 다 읽히는 분량 (1,800~2,500자)
