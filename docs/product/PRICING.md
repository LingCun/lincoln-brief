# Pricing · Roadmap · KPI

> 가격 책정·Phase 별 로드맵·핵심 지표. 비전·차별화는 [STRATEGY.md](STRATEGY.md).

---

## 1. 가격 책정

| 단계 | 가격 | 비고 |
|---|---|---|
| Phase 1 (0~6mo) | 전부 무료 | 구독자·트랙 빌딩 |
| Phase 2 (6~12mo) | 베타 월 **4,900원** 평생가 | 얼리버드, 100명 한정 |
| Phase 3 (12mo+) | 정상가 월 **9,900원** | 신규 가입자부터 |
| Phase 4 (24mo+) | 프리미엄 월 **19,900원** 옵션 추가 | 아카이브 + 1대1 Q&A |

**왜 월 9,900원**:
- 한국 paid newsletter 시장 진입가 (Substack 한국 작가 평균)
- 커피 1잔 가격 = 결제 저항 최소
- 1,000명 × 9,900원 × 90% (수수료/Stibee 후) ≈ 월 890만원 잠재 매출

## 2. 로드맵 (4 Phase)

### Phase 1 — Foundation (지금 ~ 6개월)
**목표**: 무료 구독자 300명, 발송 일관성 95%

- [x] 사이트 UI 카테고리 식별성 강화 (2026-05-19 완료)
- [ ] Stibee 무료 구독 폼 사이트 연결
- [ ] 첫 무료 뉴스레터 발송 (테스트 → 정식)
- [ ] 자동화 발행 시스템 구축 (Claude CLI + Task Scheduler)
- [ ] 휴장일 자동 감지 + 발송 스킵 처리
- [ ] 발송 모니터링·실패 알림

상세 실행 항목은 [LAUNCH.md](LAUNCH.md).

### Phase 2 — Differentiation (6~12개월)
**목표**: 무료 1,000명, 베타 유료 50명

- [ ] **워치리스트 섹션 도입** (메일에만 — 사이트는 30일 지연)
- [ ] 무료/유료 콘텐츠 분리 적용 (frontmatter `freeBody` / `paidBody`)
- [ ] 사이트에 "구독 시 더 보기" CTA 위치 최적화
- [ ] 베타 유료 구독 시작 (Stibee 결제 모듈 사용)
- [ ] 사업자등록 + 통신판매업 신고 검토

### Phase 3 — Scale (12~24개월)
**목표**: 유료 200명, 월 매출 200만원

- [ ] 정상가 월 9,900원 전환 (신규)
- [ ] 마케팅 채널 활성화 — 블로그 SEO, 트위터/스레드, 카카오 채널
- [ ] 구독자 인터뷰 (10명) → 가치 검증
- [ ] 리퍼럴 프로그램 (친구 초대 1개월 무료)

### Phase 4 — Expansion (24개월+)
**목표**: 월 매출 500만원+

- [ ] 자체 결제 시스템 (Toss/Stripe) — Stibee 한계 극복
- [ ] 웹 멤버십 페이월 — 아카이브 완전 접근권
- [ ] 사이드 상품: PDF·강의·연간 분석 책
- [ ] 프리미엄 티어 (월 19,900원) — 1대1 Q&A, watchlist 알림
