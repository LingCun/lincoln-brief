## 1. 한 줄 비전

**매일 아침 6시, 미국·한국 증시의 핵심 5분을 받는 매거진형 뉴스레터.**

포지셔닝 — 시장의 빈자리:
```
미라클레터 (무료, 깊이 얕음)
     ↓
  ★ Lincoln Brief ★  ← 깊이 있는 분석 + 매일의 워치리스트
     ↑
텔레그램 종목방 (유료, 무책임)
```

---

## 2. 타겟 사용자

| 항목 | 내용 |
|---|---|
| 연령 | 30~50대 |
| 직군 | 직장인 (출퇴근 시간 소비), 자영업자 (오전 정보 정리) |
| 투자 경험 | 1~10년 (입문 후 안정기) |
| 자산 규모 | 5천만~5억 |
| 핵심 통증 | 정보 과다 / 시간 부족 / "그래서 뭐?"가 빠진 분석 |
| 결제 의향 | 월 9,900원 = 커피 1잔 = 진입 저항 낮음 |

**사용 시나리오**: 출근 지하철에서 메일 1통 5분 안에 다 읽음 → 시장 시작 전 머릿속 정리 완료.

---

## 3. 가치 제안 (Value Proposition)

뉴스레터가 결제받는 3대 가치를 모두 충족:

| 가치 | Lincoln Brief 구현 |
|---|---|
| **시간 (Timing)** | 06:00 KST 메일 도착 — 시장 시작 전 |
| **깊이 (Depth)** | 데이터 + 분석 + 결론 + 시사점 + 필자 의견 |
| **편의 (Convenience)** | 자동 도착 → 매일 루틴화 |
| **신뢰 (Trust)** | "Lincoln의 워치리스트" 트랙 레코드로 축적 |

---

## 4. 무료/유료 분리 모델

**핵심 원칙**: 무료 사용자에게도 가치를 주되, 결제 동기가 명확해야 함.

| 구분 | 무료 (사이트) | 유료 (메일) |
|---|---|---|
| 도착 시점 | **30일 지연 공개** | **06:00 즉시** |
| 데이터 표 | ✓ | ✓ |
| 분석 본문 | 첫 1/3 (문제 제기까지) | ✓ 전체 |
| 결론 / 시사점 | ✗ | ✓ |
| Lincoln의 한 마디 (필자 의견) | ✗ | ✓ |
| **Lincoln의 워치리스트** | ✗ | ✓ |
| 다음 30일 관찰 일정 | ✗ | ✓ |

**무료 사용자도 결국 받음 (30일 후 풀버전 사이트 공개)** — SEO·아카이브·신규 유입 채널 유지 효과.

상세 분리 규칙은 [CONTENT_MODEL.md](CONTENT_MODEL.md).

## 5. 차별화 무기

다른 한국 뉴스레터 대비 Lincoln Brief 만의 자산:

1. **자동화로 매일 06:00 정확 발송** — 다른 작가는 일관성 깨짐
2. **4개 카테고리 통합 커버** — 시황·종목·예측·이슈 동시 (대부분 한두 영역만)
3. **미국 + 한국 동시** — 미국만 or 한국만 보는 메일이 일반적
4. **매거진/에세이 톤** — 분석 글의 90%는 건조함. 인문적 톤은 희소
5. **인포그래픽 썸네일** — 텍스트 위주 뉴스레터 대비 시각적 자산

## 6. 리스크 등기부

| 리스크 | 가능성 | 영향 | 대응 |
|---|---|---|---|
| 발송 일관성 깨짐 (작가 번아웃·휴가·자동화 오류) | 중 | 높 | 자동화 + 모니터링 + 휴장일/공휴일 표 사전 등록 |
| 워치리스트 트랙 레코드 부진 (수익률 누적 부진) | 중 | 높 | "추천 아닌 관찰" 톤 유지 + 손절가 명시 + 정직한 회고 |
| **법적 — 투자자문업 등록 의무** | 낮 | 매우 높 | 추천 톤 절대 금지, 면책 명시 ("정보 제공 목적"), 변호사 자문 |
| 구독자 정체 (300명에서 멈춤) | 높 | 중 | 마케팅 채널 다양화, 콘텐츠 형식 실험, 인터뷰 진행 |
| Stibee 무료 한계 (501명+ 비용) | 중 | 낮 | 첫 비용은 유료 전환분으로 자연 흡수 |
| 컨텐츠 자동화 품질 저하 | 중 | 높 | 사람 검토 단계 필수 (memory: 자동 발행 ❌) |
| 경쟁 뉴스레터 등장 | 중 | 중 | 톤·자동화·트랙 레코드는 모방 어려움 — 장기 우위 |

## 7. 의사결정 원칙

**무엇을 결정할 때 이 순서로 생각**:

1. **장기 신뢰 > 단기 트래픽** — 트래픽 위해 자극적 제목 ❌
2. **품질 일관성 > 다양성** — 평일 일정 톤 > 가끔 좋은 글
3. **수익화 < 구독자 가치** — 첫 1년은 결제 안 받음
4. **자동화 > 수작업** — 단, 검토는 사람이
5. **본인 의견 명시 > 중립** — "Lincoln의 한 마디"는 핵심 차별화
6. **법적 안전 > 성장 속도** — 종목 추천 금지선 항상 지킴
